How it works

This calculator compares a start date and an end date, then returns the difference in days. You can count from today or use a custom start date.

“Exclude today” is usually best for countdowns. “Include today” is helpful when you want to count both the start day and end day as part of the total.

What does “calendar breakdown” mean?

It shows the difference as years, months, and days instead of just total days.

Why does the number change depending on include or exclude?

Because including today adds 1 day to the count whenever the dates are different.

Does this account for time zones?

It uses local calendar dates only. It is intended for day-based counting, not hour-by-hour timing.
