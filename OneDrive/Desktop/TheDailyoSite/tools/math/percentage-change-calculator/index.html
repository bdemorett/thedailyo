How it works

Percentage change tells you how much a value increased or decreased relative to the original value. It is commonly used for prices, salaries, revenue, population changes, and investment returns.

Percentage change formula
(New − Original) ÷ Original × 100
Percent difference formula
|Value 1 − Value 2| ÷ ((Value 1 + Value 2) ÷ 2) × 100
When should I use percentage change?

Use it when one number clearly comes first and the second number is the updated value.

When should I use percent difference?

Use it when comparing two values without treating one as the official baseline.

Why can’t percentage change use 0 as the original value?

Because dividing by zero is undefined. If the original value is 0, percentage change cannot be calculated in the normal way.
